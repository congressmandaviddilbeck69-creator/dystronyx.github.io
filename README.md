# Developer Website

This repository hosts the developer website for my mobile applications.

The website is published using GitHub Pages and serves as the official developer website referenced in the Google Play Store listing of my apps.

## Purpose

The main purposes of this website are:

- Provide a developer website for mobile applications
- Host the app-ads.txt file required for Google AdMob verification
- Provide basic developer contact information
- Support transparency for app users

## app-ads.txt

This repository contains the app-ads.txt file used for advertising verification.

Location of the file:

https://username.github.io/app-ads.txt

Example entry:

google.com, pub-5102452792083242, DIRECT, f08c47fec0942fa0

This file allows authorized ad networks to sell advertising inventory for the apps.
